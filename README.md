#  Typescript + Karma + Webpack + Coverage + ESLint project boilerplate

## Quickstart

### Install dependencies
```
npm i
```

### Run unit tests
```
npm test
```

### Run codestyle
```
npm run lint
```

### Build dev bundle
```
npm run build-dev
```

### Build production bundle
```
npm run build-prod
```
