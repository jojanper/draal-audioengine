#  Audio Playback API
Audio playback and decoding engine.

## Quickstart

### Install dependencies
```
npm i
```

### Run unit tests
```
npm test
```

### Run codestyle
```
npm run lint
```

### Build dev bundle
```
npm run build-dev
```

### Build production bundle
```
npm run build-prod
```

### Serve dev or production bundle
```
npm run serve-build
```
